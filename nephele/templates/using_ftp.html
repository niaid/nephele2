Uploading Files to the Public Helix FTP Site

Step 1: Install an FTP Client
  • FTP client described below: Filezilla
  • Other FTP clients:
    • FireFTP (A Firefox plug-in)
    • Cyberduck
Step 2: Create a directory in ftp://helix.nih.gov/pub/
  1. In Filezilla, enter
    ftp://helix.nih.gov/pub
    in the Host field. Username and Password are not required. The application will open the folder /pub automatically.
  2. Click on "Quick Connect."
  3. In the "Remote site" pane, right-click and select "Create directory."
Step 3: Move Files To The FTP Server
  • Change to the directory just created, "N2_demo" in the picture below.
  • Drag files from "Local site" (your machine) to that remote directory.
Step 4: Submit Files to Nephele Using a URL
  1. Go to the Nephele website and select the pipeline you'd like to run.
  2. On the page that asks you to "Upload Data Files" select the "Upload via FTP" button.
  3. Copy the URL created on the FTP server into the "Upload Data" field. In this case the URL for would look like:
    ftp://helix.nih.gov/pub/N2_demo/
